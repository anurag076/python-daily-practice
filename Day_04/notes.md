# Day 04 Notes

## Concepts Learned

- Integer Division (//)
- Float Division (/)
- Input Handling
- Arithmetic Operators

## Solution Approach

1. Read two integers from input.
2. Use // for integer division.
3. Use / for floating-point division.
4. Print both results.

## Time Complexity

O(1)

## Space Complexity

O(1)

## Key Takeaway

Python provides two division operators:

- / → Returns a floating-point result
- // → Returns the integer quotient

Examples:

5 / 2 = 2.5
5 // 2 = 2

## Real-World Relevance

Understanding division operators is important when:

- Calculating network subnets
- Determining IP allocations
- Computing utilization percentages
- Processing automation metrics
